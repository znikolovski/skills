# Block: `step-process`

> **Status:** New — must be built
> **Loading phase:** LAZY
> **Category:** Content & Education

Numbered step-by-step process guide with icons. Used on product detail pages for "How to apply" and "How it works" sections. Each step has an icon, a heading, and a description.

## JavaScript Decoration

```javascript
export default async function decorate(block) {
  const rows = [...block.querySelectorAll(':scope > div')];

  const ol = document.createElement('ol');
  ol.className = 'step-process-list';

  rows.forEach((row, index) => {
    const [iconCell, contentCell] = [...row.querySelectorAll(':scope > div')];
    const li = document.createElement('li');
    li.className = 'step-process-item';

    // Step number
    const num = document.createElement('div');
    num.className = 'step-process-number';
    num.setAttribute('aria-hidden', 'true');
    num.textContent = String(index + 1);
    li.append(num);

    // Icon
    const img = iconCell?.querySelector('img');
    if (img) {
      img.alt = ''; // Decorative — heading provides meaning
      const iconDiv = document.createElement('div');
      iconDiv.className = 'step-process-icon';
      iconDiv.setAttribute('aria-hidden', 'true');
      iconDiv.append(img);
      li.append(iconDiv);
    }

    // Content
    const contentDiv = document.createElement('div');
    contentDiv.className = 'step-process-content';

    const heading = contentCell.querySelector('h2, h3, h4');
    if (heading) {
      const h3 = document.createElement('h3');
      h3.className = 'step-process-title';
      h3.innerHTML = heading.innerHTML;
      contentDiv.append(h3);
    }

    const desc = contentCell.querySelector('p');
    if (desc) {
      desc.className = 'step-process-description';
      contentDiv.append(desc);
    }

    li.append(contentDiv);
    ol.append(li);
  });

  block.replaceChildren(ol);
}
```

## Accessibility

- `<ol>` provides the semantic numbered list structure — screen readers announce "step 1 of 4" etc.
- Step number circles are `aria-hidden="true"` — the `<ol>` conveys order semantically
- Icons are `aria-hidden="true"` and `alt=""` — heading text provides meaning
